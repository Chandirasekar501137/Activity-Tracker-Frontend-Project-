\# Activity Tracker

A simple and interactive \*\*Activity Tracker\*\* built using HTML, CSS,
and JavaScript. This project allows users to manage daily tasks with
features like adding, completing, deleting, filtering, and resetting
tasks.

\-\--

\## Features

\- Add new activities - Mark activities as completed - Delete
activities - Filter tasks (All / Pending / Completed) - Reset all tasks
to pending - Progress bar showing completion status - Data persistence
using LocalStorage - Clean and professional UI

\-\--

\## Technologies Used

\- HTML5 - CSS3 - JavaScript (ES6)

\-\--

\## Project Structure activity-tracker/ │── index.html │── style.css │──
script.js │── README.md

\-\--

\## How It Works

1\. User adds a task using the input field. 2. Tasks are displayed
dynamically using JavaScript. 3. Each task can be marked as completed or
deleted. 4. Tasks are filtered based on user selection. 5. Data is
stored in LocalStorage so it persists after refresh. 6. Progress bar
updates automatically based on completed tasks.
